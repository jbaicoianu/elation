# dispatchEvent(ev)

Fire an event on this element. If ev.bubbles is true the event walks up through parent nodes, with each ancestor receiving a cloned event whose target stays this element but whose currentTarget is the ancestor. A method named on<type> on this element, if present, runs as a default handler before registered listeners.
