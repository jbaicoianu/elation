Data, bound to UI

Collections hold the data; UI elements render it. The list and grid below both point at the same <collection-simple> — adding an item updates both views, no manual re-render. The right column shows the framework's own events firing in real time.

// JS
const items = elation.elements.create('collection-simple', {
  items: [{ name: 'Alpha' }, { name: 'Bravo' }, { name: 'Charlie' }]
});

// Markup — both views bound to the same collection
<ui-list collection="items"></ui-list>
<ui-grid collection="items"></ui-grid>

items.add({ name: 'Delta' });    // both views update
